# News App

A simple news application built with React and Bootstrap.  
This project was created as a practice project by following a project-based learning approach.

## Features

- Fetches news data from an external API
- Displays news articles in a clean and responsive layout
- Component-based structure using React
- Styled with Bootstrap

## Technologies Used

- React
- Bootstrap
- JavaScript (ES6+)
- News API

## Project Structure

The project follows a simple and readable structure:

- `src/components` – Reusable UI components
- `src/App.js` – Main application component
- `src/index.js` – Application entry point

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
